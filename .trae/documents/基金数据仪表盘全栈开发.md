## 项目概述

基于原型图开发完整的基金数据仪表盘，使用Next.js+TypeScript+Bun前端 + Python FastAPI后端 + JSON文件存储。

## 技术栈

* **前端**: Next.js 14 (App Router) + TypeScript + Tailwind CSS + Bun

* **后端**: Python FastAPI + Uvicorn + Pydantic

* **图表**: Lightweight Charts (TradingView)

* **数据获取**: SWR

* **存储**: JSON文件（带文件锁）

## API接口设计

| 方法  | 路径                         | 描述    |
| --- | -------------------------- | ----- |
| GET | /api/funds                 | 基金列表  |
| GET | /api/funds/{code}          | 基金详情  |
| GET | /api/funds/{code}/chart    | 走势图数据 |
| GET | /api/funds/{code}/holdings | 重仓股数据 |
| GET | /api/funds/{code}/flow     | 资金流向  |
| GET | /api/market/indices        | 大盘指数  |
